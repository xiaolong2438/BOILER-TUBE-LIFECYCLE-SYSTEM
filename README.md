# BOILER-TUBE-LIFECYCLE-SYSTEM

## 锅炉受热面全寿命管理系统 v7.3 (湛江电厂稳定版)

[![Version](https://img.shields.io/badge/version-7.3-blue)]()
[![License](https://img.shields.io/badge/license-MIT-green)]()

### 📋 项目简介

本系统是一款专为火力发电厂设计的**锅炉受热面全寿命周期管理工具**，采用纯前端单文件架构，无需后端服务器即可运行。系统实现了从管段编码生成、数据录入、寿命跟踪到 AI 趋势分析的全流程管理功能。

**适用场景**：宝钢湛江自备电厂 #1机组 600MW 超超临界Π型锅炉

---

### ✨ 核心功能

#### 1. **智能编码生成器**
- 采用 6 段层级组合法：`炉号 - 系统 - 区域 - 屏/排 - 管圈 - 分段`
- 示例编码：`1-WW-FR-008-0003-000`
- 支持水冷壁 (WW)、过热器 (SH)、再热器 (RH)、省煤器 (ECO) 等系统

#### 2. **精准查询系统**
- 支持完整/局部管码模糊匹配
- 快速定位管段位置与历史数据

#### 3. **受热面部件库**
覆盖 8 大类部件：

| 部件 | 系统代码 | 规格 | 材质 |
|------|----------|------|------|
| 水冷壁 | WW | Φ63.5×7.5 | SA-210C |
| 低温过热器 | LSH | Φ57×6 | 15CrMoG/12Cr1MoVG |
| 全大屏过热器 | PSH | Φ51×7 | 12Cr1MoVG/T91 |
| 高温过热器 | HSH | Φ51/Φ54 | T91/TP304H |
| 低温再热器 | LRH | Φ63.5×4/6 | SA-210C/15CrMoG |
| 高温再热器 | HRH | Φ60×4 | T91/TP304H |
| 省煤器 | ECO | Φ51×6 | SA-210C |

#### 4. **寿命跟踪与时间线**
- 记录测厚、金相分析、无损检测、割管取样等事件
- 自动计算剩余寿命与健康度评估

#### 5. **数据录入与管理**
- **单条录入**：新增寿命事件、管段更换登记
- **批量操作**：CSV 模板下载、导入导出、数据清空
- **本地存储**：使用 localStorage 持久化数据

#### 6. **全量台账生成**
- 一键生成 1 号炉/2 号炉/全厂合并 CSV
- 支持 5139 个管理单元的快速导出

#### 7. **AI 趋势分析** 🤖
- **线性回归算法**：基于最小二乘法预测减薄趋势
- **剩余寿命预测**：自动计算 RUL (Remaining Useful Life)
- **智能诊断报告**：生成专业分析报告
- **可视化图谱**：真实数据驱动的趋势曲线

#### 8. **高精度炉管分布图** ◉
- Π型锅炉三维布局 SVG 可视化
- 交互式点击查看管排参数
- v7.3 修复：移除 CSS scale 变换，解决鼠标悬停抖动问题

### 💾 数据格式

#### CSV 导入模板
```csv
数据分类，日期，管段编码，事件类型/更换原因，关联新管码，详细描述/结论
事件，2022-04-15,1-WW-FR-015-0001-000，测厚,,标高 18m 处壁厚 7.1mm
事件，2024-05-20,1-WW-FR-015-0001-000，测厚,,标高 18m 处壁厚 6.9mm
更换，2025-10-12,1-WW-FR-015-0001-000，爆管更换，1-WW-FR-015-0001-000-R1，因减薄超标更换
```

      "ts": 1728691200000
    }
  ]
}
